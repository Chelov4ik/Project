Task Manager API

Цель проекта — облегчить управление задачами в компании, предоставляя четкую структуру для назначения задач, их отслеживания и обновления статусов. Система помогает распределять обязанности между сотрудниками, поддерживает прозрачность рабочих процессов и уведомляет пользователей о дедлайнах задач.

Описание
Этот проект представляет собой API для управления задачами в компании, созданный на ASP.NET. Приложение поддерживает три роли пользователей: Admin, Manager и Worker, каждая из которых имеет различные права доступа к функциональности системы.

Функциональность

Admin:
-Управление задачами: создание, редактирование, удаление задач.
-Управление пользователями: добавление, повышение до менеджера или понижение до рабочего, удаление пользователей.

Manager:
-Управление задачами: создание и редактирование задач.
-Просмотр списка задач и их статусов.
-Просмотр списка рабочих и их статусов.

Worker:
-Просмотр списка задач.
-Изменение статуса задач (например, выполнено, в процессе).


Что в планах:
-добавить уведомление на email за определенное времени до deadline  
-добавить чат между company и отдельно между отделами

Технологии
Backend: ASP.NET Core
Frontend: React 
Аутентификация: JWT (Bearer Tokens)
Базы данных: SQL Server
